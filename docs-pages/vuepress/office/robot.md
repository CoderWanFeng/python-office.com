## 2、常见问题

如何实现换行效果？代码如下。👇

```python
# pip install PyOfficeRobot
# 建议使用清华大学的仓库，教程：https://www.bilibili.com/video/BV1SM411y7vw/
import PyOfficeRobot

PyOfficeRobot.chat.send_message(who='程序员晚枫', message='你好' + '{ctrl}{ENTER}' + 'hello')
```
