## 代码演示

运行以下代码之前，请一定要升级一下python-office。

升级命令：pip install python-office -U，不会用升级命令的同学，去看一下第3讲的最后：[打开第3讲](https://www.bilibili.com/video/BV1m14y1y76g/)

```
import office

office.wechat.chat_robot(who='每天进步一点点')  # 智能聊天
```
