## 代码演示

运行以下代码之前，请一定要升级一下python-office。

升级命令：pip install python-office
-U，不会用升级命令的同学，去看一下第3讲的最后：[打开第3讲](https://www.bilibili.com/video/BV1m14y1y76g/)

```python
import office

res = office.video.txt2mp3(content='程序员晚枫',
                           file=r'test_files/50-12-txt2mp3/程序员晚枫.txt',
                           mp3=r'test_files/50-12-txt2mp3/程序员晚枫.mp3',
                           speak=True)
print(res)
```

视频里有一点没提到，这里给大家补充一下：这一讲的代码，是有返回值的，也就是上面代码中的``res``。

返回值代表的是mp3存储的位置，可以打印出来。
